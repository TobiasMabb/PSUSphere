\# PSUSphere



\## Description

PSUSphere is a Django-based web application for managing student organizations, colleges, programs, and memberships.



\## Features

\- Manage Colleges

\- Manage Programs

\- Manage Organizations

\- Manage Students

\- Manage Organization Memberships

\- Django Admin customization

\- Fake data generation using Faker
